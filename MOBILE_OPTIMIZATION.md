## 🚀 部署到 Netlify

### 快速部署步骤

1. **本地验证**
   ```bash
   npm install
   npm run build
   npm run preview
   ```

2. **连接GitHub仓库** (推荐方式)
   - 在 [Netlify](https://app.netlify.com) 创建新站点
   - 连接GitHub仓库
   - 设置环境变量
   - 自动部署

3. **或使用Netlify CLI**
   ```bash
   npm install -g netlify-cli
   netlify deploy --prod
   ```

4. **设置环境变量**
   在Netlify Dashboard中设置：
   ```
   VITE_GEMINI_API_KEY=your_key
   VITE_APP_URL=your-site.netlify.app
   ```

## 📱 手机使用指南

### iOS 用户
1. 打开页面，允许相机权限
2. 允许陀螺仪权限（iOS特有）
3. 拍照或上传照片
4. 点击"確認開始生日儀式"
5. 欣赏3D蛋糕（可拖动旋转）
6. 平放手机，等待3.5秒自动熄灭蜡烛
7. 打开生日贺卡

### Android 用户
1. 打开页面，允许相机权限
2. 拍照或上传照片
3. 点击"確認開始生日儀式"
4. 欣赏3D蛋糕（可拖动旋转）
5. 平放手机，等待3.5秒自动熄灭蜡烛
6. 打开生日贺卡

### 桌面用户
1. 上传照片（相机不可用）
2. 点击"確認開始生日儀式"
3. 使用鼠标旋转蛋糕
4. 点击"電腦測試/手動吹蠟燭"按钮手动熄灭蜡烛
5. 打开生日贺卡
